# Voronoi Images

Implementation of the Bowyer-Watson algorithm that generates a voronoi diagram from an image.

### Made with:
* Python 3.6
* OpenCV 3.4
* Numpy

### Usage:
```
python3 voronoi.py path_to_image number_of_points
```

For example:
```
python3 voronoi.py img.jpg 2000
```
